## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm
  ```sh
  npm install -g @angular/cli
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/abrahamcastelancruz/jaque.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app in localserver ðŸŽ‰
   ```sh
   ng serve
   ```
